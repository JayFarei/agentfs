# AWS

AWS-authored materials for the hackathon. AWS is a required component of the finalist stack (top-6 finalists must build on **MongoDB Atlas + AWS**), so the participant guide here is load-bearing.

| File | What it is |

## When to read

- Before signing up for AWS for the hackathon, to know what services qualify under Free Tier and what the gotchas are
- When choosing your AI service (Bedrock vs other AWS AI offerings) so you don't accidentally exit Free Tier mid-build
- If your project deploys long-running infrastructure, double-check that it stays inside the Free Tier limits called out in the guide
