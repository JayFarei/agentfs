# Quickstarts

Hands-on, copy-and-run starting points for hackathon teams who need to be productive on Atlas in under an hour. The resources here cover three foundational moves: connecting Python to Atlas, generating high-quality embeddings with Voyage AI, and running your first vector query in Atlas. Pair these with the deeper material in `01-key-resources/` once the basics work end-to-end.
