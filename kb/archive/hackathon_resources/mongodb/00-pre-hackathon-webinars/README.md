# Pre-Hackathon Enablement

A 5-part webinar series MongoDB ran in late 2025 / early 2026 ahead of dot local San Francisco and the Agentic Evolution Hackathon. Each session is ~50-60 min, hosted on YouTube, with a companion MongoDB resource page and an end-of-session Skill Badge check.

## Progressive discovery

1. **Just orienting yourself?** Read [`00-series-overview.md`](./00-series-overview.md), 30 seconds.
2. **Need a specific concept?** Pick the file below by topic, scan its TL;DR + Key Takeaways (~1 min).
3. **Want to use it in a build?** Read the "What's Covered" section (~3 min).
4. **Need to verify or quote precisely?** Open the linked source video / page from the file's footer.

## Notes on capture

- **Sources used:** MongoDB's curated resource page for each webinar (defuddle), YouTube auto-generated transcripts via yt-dlp (W1, W2, W5 captured cleanly; W3 hit YouTube rate-limit, W4 has no public YouTube link in the hackathon guide), plus video metadata.
- **Speakers identified from transcripts:** W1 + W5 hosted by Aaron Becker (Curriculum Engineer, MongoDB University). W2 hosted by Oscar Han (Solution Architect, MongoDB Netherlands).
- **Authoritativeness:** The "What's Covered" sections on W1, W2, W5 are paraphrased from the actual session transcripts. W3 and W4 sections rely on MongoDB's curated resource page only and should be treated as a directional summary, dive into the video for concrete examples.
- **Skill badges:** Each session ends with a 10-question check at `mdb.link/<topic>-sf`. Earning all five badges completes the Modern Data Architecture Mastery track.
