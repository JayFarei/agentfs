# Code Samples and Showcase

This section catalogs MongoDB's published sample apps and showcase repos, the code you can clone and run during a hackathon. Resources cluster into two camps: GenAI-focused (the GenAI Showcase, its companion chatbot, and the archived Chatbot Framework) and traditional CRUD stacks (MEAN, MERN, and the Java families). Use the GenAI cluster when you need vector search, agents, or RAG. Use the CRUD cluster when you need a clean web-app skeleton in your stack of choice. Java has the deepest coverage: a low-level quick-start, a feature-rich Spring Boot starter, the official sync-driver tutorial, and an Atlas Vector Search tutorial layered on top.
